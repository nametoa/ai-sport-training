## 步骤一：建立教练角色

```
亲爱的教练，

我希望你担任我的【混合训练教练（Concurrent Training Specialist）】，
同时兼顾：专业跑步教练 + 力量训练教练 + 运动营养顾问。

我的训练模式是混合训练（Concurrent Training）—— 同时进行力量训练和跑步训练。
你的任务是：在保证健康、避免伤病的前提下，合理安排力量和跑步的训练顺序与强度分配，
让我以最佳状态完成下面的比赛目标。

在制定任何计划之前，请先充分了解我的背景、数据和现实情况。
```

## 步骤六：让 AI 主动提问

```
在制定计划之前，请向我提出 10 个你认为最重要的问题，按以下分类：
- 比赛目标（特别是九华山的完赛策略和无锡的配速计划）
- 混合训练偏好（力量和跑步的优先级）
- 伤病与恢复
- 生活方式与营养
- 赛间恢复策略
```

## 步骤七：生成训练计划

```
请根据以上所有信息，为我创建一个针对两场背靠背比赛的训练计划：

第一阶段（2/25-3/7）：九华山赛前减量
- 维持跑感但降低跑量
- 力量训练减容量（-40%组数），保持强度
- 越野专项（坡度训练）

第二阶段（3/8）：九华山比赛日
- 九华山南北穿越 40km 比赛策略与配速方案

第三阶段（3/9-3/21）：恢复 + 无锡赛前调整
- 前 3-4 天：主动恢复（步行、拉伸、轻松骑行）
- 中期：逐渐恢复跑步，测试腿部状态
- 后期：1-2 次马拉松配速试跑
- 力量训练：极轻/激活为主

第四阶段（3/22）：无锡马拉松比赛日
- 全马配速方案（目标 2:55-2:59）

第五阶段（3/23-4/5）：赛后恢复
- 逆减量恢复法
- 重新建立混合训练节奏

要求：
- 精确到每天的训练安排（中午力量 + 晚上/下午跑步的双训日安排）
- 说明每个安排的原因
- 标注力量训练与跑步的最小间隔时间
- 控制每周总训练负荷在 COROS 建议范围内
```
