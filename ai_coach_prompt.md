## 步骤一：建立教练角色

```

我希望你担任我的【专业跑步教练 + 力量训练教练 + 运动营养顾问】。

你的任务是：在保证健康、避免伤病的前提下，让我以最佳状态完成我的下一个跑步目标。

在为我制定任何训练或营养计划之前，我希望你先充分了解我的背景、习惯和现实生活情况。
```

## 步骤六：让 AI 主动提问（核心差异化步骤）

```
- 比赛目标

- 生活方式

```

## 步骤七：生成训练计划

```
今天是 202-02-25
计划 ：2026-03-08  九华山南北穿越 40km 爬升 4000 计划一日完成，  2026-03-22 无锡马拉松全马 目标破三

请根据以上所有信息，为我创建一个四阶段训练周期：
1）基础阶段：建立有氧耐力，保持一致性，预防伤病
2）进阶阶段：加入间歇、节奏跑、坡跑，逐步增加长距离
3）减量阶段：减少跑量，保持强度，为比赛蓄能
4）恢复阶段：赛后恢复与重新适应

要求：
- 说明每阶段目标和周里程变化
- 整合力量训练（每周2-4次，不安排在关键跑课前一天）
- 整合灵活性训练（针对脚踝、髋、腘绳肌）
- 说明每个安排的原因
```
